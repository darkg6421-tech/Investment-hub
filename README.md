# Investment Hub

## Project Description
Investment Hub is a comprehensive platform designed to assist users in managing and calculating their investment portfolios effectively. The application offers various tools and calculators to help make informed financial decisions.

## Features
- Portfolio management
- Investment calculators (including ROI, compound interest, etc.)
- User-friendly interface for easy navigation
- Data visualization of investment performance over time

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/darkg6421-tech/Investment-hub.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Investment-hub
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage Guide
- To start the application, run:
   ```bash
   npm start
   ```
- Follow the on-screen instructions to manage your investments and utilize the calculators.

## Calculator Explanations
- **ROI Calculator**: Calculates the return on investment based on the initial amount and final amount.
- **Compound Interest Calculator**: Computes the amount of compound interest accrued over time based on the principal, interest rate, and time period.
